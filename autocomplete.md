
# Autocomplete

Learn how to create a Autocomplete using Styled Components

1. Create a input component

```javascript
import styled from 'styled-components';

const Input = styled.input`
  border: 1px solid #d6d6d6;
  display: block;
  margin-bottom: 20px;
  padding: 10px 15px;
`;

export default Input;
```
